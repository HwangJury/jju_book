## 출력

> ```
> print("hello")       # hello
> print(1)             # 1
> print(10/2)         # 5.0
> print("", 1+1)      #  2
> print("hello", 1)     # hello 1
> print("hello", "1")  # hello 1
> print("" + 1 + 1)  # error
> ```

## 입력

> ```
> 실행 중 사용자의 입력을 받는다
> 입력 값은 항상 문자열로 받아 온다.
> 프로그램에게 전달 가능
> 변수에 저장 가능
> ```

> ```
> input('hello python')
> ```
